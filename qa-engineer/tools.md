# Tools: QA Engineer

Stack preferencial para QA pragmático no ecossistema atual.

## Testes Automatizados
- pytest
- pytest-cov
- pytest-xdist

## Testes de API
- Postman
- Newman
- httpx ou clientes Python leves quando o contexto pedir automação em código

## Segurança Básica
- Bandit

## Performance e Diagnóstico
- k6
- logs da aplicação e observabilidade disponível no ambiente

## Relatórios
- JUnit XML ou relatórios simples consumíveis em CI

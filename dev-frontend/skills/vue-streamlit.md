# Skill: Vue + Streamlit

## Quando usar
Use esta skill para decidir e implementar interfaces com Vue ou Streamlit de forma pragmática.

## Padrão Sênior
- Use Streamlit quando a prioridade for velocidade, clareza e fluxo orientado a dados.
- Use Vue quando a interface precisar de rotas, estados mais sofisticados e componentes reutilizáveis.
- Não replique em Vue o que um app Streamlit resolveria com menos custo.
- Não force Streamlit em cenários que exigem interação rica e composição avançada.

## Boas Práticas
- Fazer a escolha da stack explicitamente.
- Manter a integração com API simples e previsível.
- Tratar estados de erro, vazio e loading como parte da entrega.
- Priorizar legibilidade e manutenção.

## O que evitar
- Escolha de stack por preferência pessoal.
- Misturar paradigmas de forma arbitrária.
- Interface com muita abstração e pouca clareza.
- Acoplamento do frontend ao formato incidental do backend.

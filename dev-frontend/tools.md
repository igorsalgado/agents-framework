# Tools: Dev Frontend

Stack preferencial para frontend pragmático.

## Frameworks Principais
- Streamlit
- Vue 3
- Vite

## UI e Estilo
- Tailwind CSS
- Headless UI ou bibliotecas equivalentes quando acelerarem entrega
- VueUse para utilitários recorrentes

## Estado e Navegação
- Pinia
- Vue Router
- Axios ou `fetch` com camada fina de integração

## Dados e Visualização
- Plotly
- ECharts ou bibliotecas leves quando fizer sentido

## Testes e Qualidade
- Vitest
- Vue Test Utils
- Playwright apenas para jornadas realmente críticas
